 헤더 파일 추가

### 문자열 &문자 

"*" 문자열

'*' 문자 

 C에서 printf의 포맷팅 문자열 사용
– E.g., “%6d”
• setw: 오른쪽 정렬된 출력 폭(width) 지정

```c++
for (int i = 1; i <= 1000; i *= 10 )
cout << setw( 6 ) << i << endl;
```

– setw는 그 효과가 1회에 그친다.

```c++
cout << setw( 6 );
for (int i = 1; i <= 1000; i *= 10 )
cout << i << endl;
```





## 문자열 입력  소문자 string

#### 공백 포함 문자열 getline(cin, string--)

```c++
#include <iostream>
#include <string>
using namespace std;
int main()
{
string mystr;
cout << "What's your name? ";
getline(cin, mystr);
cout << "Hello " << mystr << ".\n";
cout << "What is your favorite team? ";
getline(cin, mystr);
cout << "I like " << mystr << " too!\n";
return 0;
}
```
