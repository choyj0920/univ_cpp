## 이차원 배열 

```c++
int [3][5] num;

//간략표현  
num [3][5] =&num[0];
num[0] =&num[0][0];
num[2] =&num[2][0];
num[0][0] = 0;

//포인터 표현
num[2][3] =*(*(num+2)+3)
```

```c++
//이차원 배열은 선언할 떄 초기화 하더라도 가장 고차원의 정보만 생략 할수 있다
int s[][3]={
```
